# FaceDocVerify

Sistema Seguro de Verificación de Identidad utilizando OCR Open Source y Reconocimiento Facial con Mecanismos de Detección de Fraude.

## Descripción

FaceDocVerify es una plataforma desarrollada para validar la identidad de usuarios mediante:

* Extracción de datos de DNI utilizando OCR.
* Comparación biométrica facial entre documento y selfie.
* Validación de mayoría de edad.
* Validación de vigencia del documento.
* API REST segura protegida mediante JWT.
* Auditoría y trazabilidad de solicitudes.

## Tecnologías

* Python 3.11
* FastAPI
* OpenCV
* Tesseract OCR
* EasyOCR
* DeepFace
* JWT
* Git
